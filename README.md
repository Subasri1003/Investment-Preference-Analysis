# Investment-Preference-Analysis

## Project Overview
This project analyzes customer investment preferences to understand their behavior and decision patterns.

## Objective
- Identify popular investment options
- Analyze risk appetite of customers
- Understand factors influencing investment decisions

## Tools Used
- Excel (Pivot Tables, Charts)
- Data cleaning & Analysis
- Visuallized data using charts

## Steps Performed
- Cleaned raw dataset
- created pivot tables for analysis
- Visualized data using charts

## Key Insights
- Majority prefer low-risk investments
- Fixed deposits and gold are popular choices
- Age group 25-40 shows higher investment activity
